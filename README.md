## Example ReasonML Monorepo (StarterKit)

### How to start
- clone this project 
- go to directory of project `cd re-monorepo-t`
- install dependencies `yarn install`
- start server `yarn server` (default port: be(3000), fe(8000))
- enjoy
   
### Packages
- @ri7nz/fe (frontend)
- @ri7nz/be (backend)

### Stack
- react
- reason-react
- express
